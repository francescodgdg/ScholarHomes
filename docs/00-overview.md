## Quick Summary for Francesco

### What We're Building (MVP)

**For Students:**
1. Sign up with .edu email → select university
2. Browse listings with filters (price, rooms, dates, type)
3. Post sublet or rental listings with photos
4. Message other students in-app
5. Manage your own listings

**For Admin (You):**
- Dashboard to see all listings and users
- Approve, edit, pause, or delete any listing
- Ban users if needed
- Add listings manually on behalf of someone

### Design Direction
- **Clean & Simple** - Easy for students to use quickly
- **Marketplace-style** - Similar to Airbnb/Facebook Marketplace
- **Trust-focused** - .edu verification, clear user profiles
- **Mobile-first** - iOS and Android via React Native

- **Backend:** Supabase (PostgreSQL + Auth + Storage)
